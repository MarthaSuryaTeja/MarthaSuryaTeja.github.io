About

The self-balancing robot is an autonomous two-wheeled system powered by an Arduino microcontroller, which processes data from an MPU accelerometer to maintain balance. It uses stepper motors driven by a CNC shield motor driver for precise movement adjustments. The HC-05 Bluetooth module allows for remote control via a smartphone or computer. The robot’s lightweight, durable 3D-printed chassis enables easy customization and prototyping. The MBot is a 6-DOF robotic manipulator designed using Fusion 360, with each joint and link carefully modeled for accuracy. The manipulator is integrated with ROS using a URDF file and simulated in Gazebo to refine its behavior. MoveIt’s motion planning capabilities enable the manipulator to perform tasks like pick-and-place operations. This project demonstrates the power of modern design tools and ROS in creating adaptive, real-world robotic systems.
