About

Vinata was inspired by the Pepper and Mitra robots available in the market, with the core concept focused on creating an interactive guest greeting system. The robot's operation is primarily controlled by a Raspberry Pi, which handles all the functions, while the Pysttx3 library is used to convert text to speech. An Arduino is responsible for controlling the servo motors, receiving commands from the Raspberry Pi via serial communication. A 16x4 keypad matrix allows users to interact with the robot, start its functions, and use special keys to address specific guests during events. Additionally, a barcode scanner is integrated to retrieve student data from a database, enabling the robot to greet students and mark attendance.
